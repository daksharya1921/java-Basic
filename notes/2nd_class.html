🎯 Topics Covered

  • Java class naming convention

  • Java class structure

  • Reading compiler errors

🧠 Key Concepts

Class Naming: Start every class with a capital letter. Multi-word class names use PascalCase.

class Student
class PaymentProcessor
class StudentDetails

Main Method

public static void main(String[] args){

}

Compilation

javac Payment.java

Execution

java PaymentProcessor

💻 Sample Program

class PaymentProcessor {

    public static void main(String[] args) {
        System.out.println("Processing Payment");
        System.out.println("Payment has been processed");
    }

}

❓ Doubts / Questions

  • Does every class need a main() method? → No.
  • Why line numbers? → Helps locate compilation errors.
  • Why recompile? → JVM executes the latest bytecode only after recompilation.

📌 Important Points

  • Every statement ends with ';'
  • Compilation uses file name.
  • Execution uses class name.
  • Always read compiler errors carefully.
