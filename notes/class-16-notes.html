What is an Array? (opening only)

The instructor opened by asking the class what an array is. The partial answer captured: an array is a continuous block of memory used to store something (elements). The full explanation, declaration syntax, and examples were not included in what was shared.

return — Exits the Method Immediately

When a method hits a return statement, it exits the method right there — no further lines in that method execute afterward. The instructor illustrated this with a conditional-processing example (see below).

Example: return Used for Conditional Early Exit

Scenario discussed: a customer provides contact details for notification.

  • If a phone number is available → send a WhatsApp message, then return (exit the method — no need to also check email).
  • If the phone number is missing → send an email instead, then return.

This shows using return not just to send back a value, but to stop further processing once the right condition/branch has been handled — avoiding unnecessary checks afterward.

Broader Lesson: Choosing return / break / continue Deliberately

The instructor emphasized this is a mindset to apply generally, not just in one exercise: before writing code, think about whether you need to let processing run end-to-end, or whether you should exit early once your goal is achieved — using return (exit method), break (exit loop/switch), or continue (skip to next loop iteration) as appropriate.

💻 Code Snippet (Reconstructed From the Described Example)

Note: this is a reconstruction of the WhatsApp/email scenario as described verbally, since no actual code was shown in the shared transcript.

private static void notifyCustomer(String phoneNumber, String email) {
    if (phoneNumber != null && !phoneNumber.isEmpty()) {
        System.out.println("Sending WhatsApp message to " + phoneNumber);
        return; // exit — no need to also send email
    }

    if (email != null && !email.isEmpty()) {
        System.out.println("Sending email to " + email);
        return; // exit after handling email
    }

    System.out.println("No contact method available");
}
